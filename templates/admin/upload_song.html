How to get Google Drive direct links
For Audio Files:
  1. Upload your audio file to Google Drive
  2. Right-click the file and select "Get link"
  3. Change sharing to "Anyone with the link can view"
  4. Copy the file ID from the URL: https://drive.google.com/file/d/FILE_ID/view?usp=sharing
  5. Use format: https://drive.google.com/uc?export=download&id=FILE_ID
For Image Files:
  1. Upload your image to Google Drive
  2. Get the file ID same as above
  3. Use format: https://drive.google.com/uc?export=view&id=FILE_ID
Important: Make sure your Google Drive files are set to "Anyone with the link can view" for them to work properly.
